Check for redundant ORDER BY when explicitly specified columns of clustered index. In this cases ORDER BY will be ignored.

CREATE TABLE T (
Id INT NOT NULL PRIMA column value - IdRY KEY,
Name VARCHAR(100) );
GO
INSERT INTO T (Id, Name)
SELECT Id, Name
FROM S
ORDER BY Name;

In this case ORDER BY will be ignored, because Id is presented and it will be sorted by clustered index.
