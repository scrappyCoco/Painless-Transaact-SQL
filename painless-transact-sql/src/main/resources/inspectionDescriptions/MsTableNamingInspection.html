Check for table name.

Available context:

  • db - database name
  • schema - schema name

Context variables could be added as named groups (see examples).

Examples:

  • UPPER_CASE: [A-Z]+([_][A-Z]+)*
  • snake_case: [a-z]+([_][a-z]+)*
  • kebab-case: [a-z]+([-][a-z]+)*
  • PascalCase: ([A-Z][a-z]+)+
  • camelCase: [a-z]+([A-Z][a-z]+)+
  • DATABASE_NAME_my_schema_MyTableName: (?<db>.+)_(?<schema>.+)_[A-Z]+([_][A-Z]+)*
